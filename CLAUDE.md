# 디하파트너즈 채용 공고 파이프라인

## 이 프로젝트가 하는 일

디자이너 채용 공고를 인스타그램 + 스레드에 올리기 위한 이미지/텍스트를 자동 생성합니다.

## 사용자가 아래 양식으로 채용 정보를 보내면 바로 실행하세요

```
기업명
-업무 시간 : 매일 X시간 / 시작시간~끝시간
-업무 시작일 : ASAP 또는 구체적 날짜
-특이사항
```

## 실행 순서

### 1. Figma MCP로 이미지 생성

- Figma 파일 키: `byXx50MB9bSu1x6bqjWmla`
- 템플릿 노드 ID: `1:2`
- 아래 순서로 실행:
  1. template(1:2) 복제
  2. 복제본에서 "variable" 텍스트 노드 찾기
  3. Noto Sans KR Bold 폰트 로드 후 텍스트를 `(기업종류 / 포지션)` 형태로 변경
  4. 복제본 이름을 기업명으로 설정
  5. viewport를 복제본으로 이동
- 사용자에게 "Figma 데스크톱에서 Export해주세요" 안내

### 2. 텍스트 생성 + txt 파일 저장

- `output/기업명.txt` 파일 하나에 아래 4개 섹션을 모두 포함:
  - `===== 인스타그램 =====`
  - `===== 스레드 메인 =====`
  - `===== 스레드 댓글 =====`
  - `===== 어드민 업로드 =====`

## 텍스트 작성 규칙

### 공통
- "디하파트너즈"는 항상 붙여쓰기 (띄어쓰기 금지)
- 모든 섹션에서 문단 사이에 빈 줄 넣기 (가독성)

### 업무 시간 표기
- 구체적 시간 있으면: `매일 4시간 9:00-14:00`
- 시간 미정이면: `매일 4시간`
- 시간 수와 실제 시간이 안 맞으면 (예: 4시간인데 9~2시=5시간): 점심시간 포함으로 판단 → `매일 4시간 9:00-14:00 (점심시간 포함)`

### 기업 종류
- 너무 특정하지 않게 일반화 (성형외과 → 미용의원, 화장품 회사 → 뷰티 브랜드 등)

### 특이사항 반영
- 특이사항은 "~~~디자이너님을 찾습니다" 설명에 자연스럽게 반영

### 스레드 댓글 지원 링크
- URL: `https://partners.designer-hire.com/apply-urgent/?utm_source=owned&utm_medium=threads&utm_campaign=dhp_now&utm_content=YYMMDD`
- utm_content는 당일 날짜 (예: 260409)

### 어드민 업로드 섹션
- 제목: 지원 분야의 / 앞 부분 (기업 종류)
- 부제목: 지원 분야의 / 뒤 부분 (포지션)
- 기업명: 입력받은 기업명
- 시간: 업무 시간

### 인스타그램 본문 템플릿

```
🚨급구
디하파트너즈에서 {업무설명} 프리랜서 디자이너님을 찾습니다.

업무 시간 : {업무시간}
업무 시작일: {시작일}

*주말/공휴일 제외
*업무시간 사전 협의 시 변경 가능

관심있는 프리랜서 디자이너님들의 많은 지원을 기다리고 있어요. 바이오의 링크를 통해 지원해주시면, 지원서 검토 후 적합한 분께 개별(카카오톡메시지 또는 문자)로 연락드리겠습니다.

📎지원방법
1️⃣ 프로필의 링크 접속
2️⃣ 지원 서류 작성
3️⃣ 지원 분야 [{기업종류} / {포지션}] 선택
4️⃣ 제출

__
🔖 지원 전에 꼭 확인해 주세요!
디하파트너즈는 프리랜서를 위한 시간제 업무 매칭 플랫폼으로, 일반 외주와 조금 달라요. 정해진 시간만 일하는 대신, 그 시간에는 온전히 하나의 기업에 집중하는 방식이에요.
* 매일 정해진 시간에 컴퓨터 앞에서 근무할 수 있어야 해요 (주말, 공휴일 제외)
* 정해진 업무 시간 동안에는 담당 기업의 업무에만 집중해요
* 기업 담당자와 실시간 채팅으로 소통하며 업무를 진행해요

{해시태그}
```

- 지원방법/확인사항/해시태그 섹션 없음

## 주의사항

- 이미지는 반드시 Figma MCP로 생성. 절대 Pillow 등 다른 도구로 대체하지 말 것.
- 이미지 로컬 저장(export)은 사용자가 Figma 데스크톱에서 직접 수행.
- txt 파일은 output/ 폴더에 기업명.txt로 저장.
